Generated release-ready files live in `dist/<platform>/`.

Telegram Cloud links like `t.me/addtheme/...` can be added after the themes are published in Telegram.

### Desktop

Download a `.tdesktop-theme` file, then open it with Telegram Desktop or select it from **Settings** → **Chat Settings** → **Choose from file**.

### Android

Open an `.attheme` link on your device. Telegram should show a theme preview with an **Apply** action.

### iOS

Download a `.tgios-theme` file, open it from Files, then choose Telegram from the system share sheet. Telegram should show a theme preview with an **Apply** action.

### macOS

Open the `.palette` file. Telegram for macOS will prompt you to apply it. If you use Telegram Desktop on Mac, download the `.tdesktop-theme` instead.

## Design

Roveliese centers on quiet surfaces, readable contrast, and a visible rose identity. Telegram's native blue/green emphasis is shifted toward rose accents where it makes sense, while status colors and media affordances stay distinct enough to scan quickly.

Each variant keeps the same Roveliese character while adjusting its contrast and temperature for a different reading environment.

## What's Covered

- Telegram Desktop theme variables
- Android theme variables
- iOS theme variables
- macOS theme variables
- Solid in-palette chat wallpapers for desktop and macOS outputs
- Dark, Light, Warm, and Storm variants
