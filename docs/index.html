All files

Desktop

Download a .tdesktop-theme file, then open it with Telegram Desktop or select it from Settings → Chat Settings → Choose from file.

Android & iOS

Open the downloaded theme file from your device. Telegram shows a theme preview with an Apply action. If the browser downloads the file instead, share it to Telegram from the system share sheet.

macOS

Open the .palette file. Telegram for macOS will prompt you to apply it. If you use Telegram Desktop on Mac, download the .tdesktop-theme instead.
